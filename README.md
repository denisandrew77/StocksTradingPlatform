📈 Trading App (Java)

A trading application built in Java, designed to simulate stock and cryptocurrency trading. 
The project demonstrates the use of multiple design patterns and clean architecture principles.

🚀 Features

👤 User Management

- Create and manage user accounts

- Track user portfolios and balances

📊 Trading System

- Buy and sell stocks

- Trade cryptocurrencies

- Maintain transaction history

💼 Portfolio Tracking

- View owned assets

- Monitor portfolio value

⚙️ Design Patterns Used

- Singleton (single stock management instance and single user management instance)
- Factory method (stocks/crypto creation)
- Decorator (owned crypto/stocks)
- Facade (market initialisation)
- Strategy (trading strategy)
- Mediator (between user and market)
